1Prepara il file JSON

Assicurati di avere il file outputfile.json con i dati delle password nel formato corretto.

2Esegui lo script di setup

Apri un terminale nella directory dell'applicazione ed esegui:

python setup_db.py
3Configura la password

Lo script ti chiederà di:

  • Specificare il percorso del file JSON (default: outputfile.json)
  • Inserire una password per cifrare il database (minimo 6 caratteri)
  • Confermare la password
4Riavvia l'applicazione

Dopo aver configurato il database, ricarica questa pagina e potrai accedere con la password impostata.

Importante: Ricorda la password che imposti durante il setup! Sarà necessaria per accedere ai tuoi dati criptati.
